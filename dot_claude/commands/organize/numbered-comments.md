description: Reorganize a config or code file into numbered sections with a matching numbered table of contents.
argument-hint: <file-path-or-glob>
---

Apply the `organize-with-comments` skill to the file(s) at $ARGUMENTS.

Read `SKILL.md` from the skill folder for the full technique. Skip the style-selection prompt and use the **Numbered** style (banner header prefixed with a section number) from the catalog for every section header, pairing it with a numbered table of contents. Write the reorganized result back to the original path and briefly summarize the sections you created.
