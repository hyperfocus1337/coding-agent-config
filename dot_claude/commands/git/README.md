# git slash commands

Shorthand command names and what they map to.

| Shorthand | Previous command | Description                                          |
| --------- | ---------------- | ---------------------------------------------------- |
| `com`     | commit           | Create a git commit                                  |
| `commu`   | commit-multiple  | Split changes into a logical sequence of commits     |
| `compu`   | commit-push      | Commit and push                                      |
| `cppr`    | commit-push-pr   | Commit, push, and open a PR                          |
| `clbr`    | clean-branches   | Delete stale local branches whose remote was deleted |
| `clwt`    | clean-worktrees  | Remove worktrees + delete their `[gone]` branches    |
| `chlog`   | changelog        | Generate a changelog for a time period               |
