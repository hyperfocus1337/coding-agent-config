# Output template

Fill in this skeleton. Drop any metadata line whose value is not present in the transcript. Use sentence case for all headings. No bullet points in the body. No em dashes.

```markdown
Meeting: <title or subject>
Date: <date>
Participants: <names and roles>
Duration: <hh:mm, if derivable>
Source language: <only if translation was needed>

## <descriptive heading for the first theme>

<Concise prose covering the key points of this theme, with speaker attribution and timestamps where relevant.>

## <descriptive heading for the next theme>

<Concise prose.>

## Decisions

<Prose listing the concrete decisions reached. If none, state that no decisions were made.>

## Action items

| Owner    | Task   | Deadline   |
|----------|--------|------------|
| <person> | <task> | <deadline> |
| <person> | <task> | <deadline> |
```

Notes:
- Use two to six themed sections, not a fixed number.
- Use "Not specified" inside table cells when a value is genuinely absent.
- If there are no action items, replace the table with: "No action items were identified."
