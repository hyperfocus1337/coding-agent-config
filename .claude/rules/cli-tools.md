### Preinstalled CLI tools

These are always available. Prefer them over slower equivalents:

- `rg` (ripgrep) for text search, `fd` for file search
- `bat` for syntax-highlighted file viewing
- `yq` / `jq` for YAML / JSON processing
- `miller` (`mlr`) and `csvkit` for CSV processing
- `httpie` (`http`) for HTTP requests
- `shellcheck` for shell script linting
- `ctags` for symbol indexing
- `ncdu` for disk usage, `lsof` for open files
- `socat` / `nc` for sockets, `strace` for syscall tracing
- `rsync` for file sync
- `moreutils` (`sponge` / `ts`) for pipeline helpers
