# Android MCP Integration

This is how you enable MCP servers in the Claude Android app

## Setup

1. Open [claude.ai](https://claude.ai) in a browser
2. Navigate to [Customize → Connectors](https://claude.ai/customize/connectors)
3. Click **Add Custom Connector** and enter the MCP server name and URL
4. The connector will then appear and work in the Android app

## Connectors

| Name     | Remote MCP Server URL          |
| -------- | ------------------------------ |
| Context7 | `https://mcp.context7.com/mcp` |
